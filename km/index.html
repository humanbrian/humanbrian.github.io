  • WebGL បង្ហាញការបង្ហាញកម្មវិធី (SwiftShader)៖ កម្មវិធីរុករកតាមអ៊ីនធឺណិតមិនមាន GPU ឬ Hardware បង្កើនល្បឿនទេ ដូច្នេះកម្មវិធីត្រាប់តាមនឹងយឺត។ បើកការបង្កើនល្បឿនផ្នែករឹង និងធ្វើបច្ចុប្បន្នភាពកម្មវិធីបញ្ជា GPU របស់អ្នកជាមុនសិន។
  • កុំព្យូទ័រយួរដៃមាន GPU ដាច់ដោយឡែក ប៉ុន្តែការពិនិត្យឃើញក្រាហ្វិករួមបញ្ចូល Intel៖ កម្មវិធីរុករកអាចប្រើ iGPU ។ ត្រូវប្រាកដថា MuMu ប្រើ GPU ដាច់ដោយឡែក (កំណត់វានៅក្នុងផ្ទាំងបញ្ជា GPU ឬការកំណត់ក្រាហ្វិកវីនដូ) ។
